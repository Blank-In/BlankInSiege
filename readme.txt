BlankInSiege 사용설명서
	명령어
		/blanksiege <tag> : 들고있는 아이템에 전용 NBT태그를 추가합니다.
		/blitzcharge <username> <count>, /blitzcharge <count> : 해당 유저의 섬광방패 충전량을 조절합니다 음수 입력시 충전량을 줄입니다.
		
	권한
		관리자 BlankInSiege.op.*
			BlankInSiege.op.blanksiege
			BlankInSiege.op.blitzcharge

		유저  BlankInMusic.user.*
		
	태그목록
		blitz : 섬광방패 (좌클릭으로 격발) /blanksiege blitz
		glowtracking : 좌클릭으로 투사체 발사 후 우클릭으로 주변의 엔티티에게 발광 효과 /blanksiege glowtracking
